# Rafael Stern Personal Website

Static personal website for Rafael Stern, prepared for GitHub Pages.

## Local Preview

Open `index.html` in a browser. No build step or package installation is required.

## Recommended GitHub Pages Setup

1. Create a public GitHub repository named `rafastern.github.io` for the cleanest URL.
2. Copy the contents of this `personal-website` folder into the repository root.
3. Commit and push to GitHub.
4. In GitHub, open **Settings > Pages**.
5. Set **Source** to `Deploy from a branch`.
6. Set **Branch** to `main` and folder to `/root`.
7. Save.

The site should publish at:

```text
https://rafastern.github.io/
```

If using a project repository instead, for example `personal-website`, publish from that repository and expect:

```text
https://rafastern.github.io/personal-website/
```

## Custom Domain Later

After buying a domain, add it in **Settings > Pages > Custom domain**. GitHub will provide the DNS records to add at the domain registrar.

## Privacy Notes

The site intentionally avoids publishing phone number, birth date, signed forms, application packets, and private job-application documents. Only public professional profile links, selected publications, and safe visual assets are included.
